HTML, CSS Y JavaScript

Son los tres pilares fundamentales del desarrollo web que trabajan juntos para crear páginas web dinámicas e interactivas.
HTML se encarga de la estructura y el contenido, CSS del diseño y la presentación, y JavaScript de la interactividad y la funcionalidad.

Definiciones

HTML (HyperText Markup Language):

Es el lenguaje de marcado que define la estructura y el contenido de una pagina web. Utiliza etiquetas para delmitar dierentes elementos, como parrafos, encabezados, imagenes, enlaces,etc.

CSS (Cascading Style Sheets):

Es un lenguaje de estilos que se utiliza para controlar la presentacion de una paginas web. Permite definir aspectos como los colores, fuentes, tamaños, margenes, etc, de los elementos HTML.

JavaScript (Program.Language):

Es un lenguaje de programacion que permite agregar funcionalidad interactiva a una pagina web. Permite manipular elementos HTML, responder a eventos del ususario realizar calculos etc.

Analogia de la web y una casa:

HTML

Estructura de la casa.

Es como los cimientos, las paredes, techos, y distribucion de las habitaciones.

HTML define la estructura basica de una pagina web: que elementos hay(titulo, parrafos, botones, imagenes, etc) pero sin preocuparse por omo se ven.

CSS

Decoracion y estilo de la casa

Es la pintura de las paredes, el tipo de piso, los muebles, las cortinas, el diseño de la cocina.

CSS se encarga de como se ve la pagina: colores, fuentes, margenes, alineaciones, estilos responsivos...lo visual.

JavaScript

Los sistemas electricos y electronicose la casa.

Es como los inerruptores que encienden luces, las puertas automaticas, el termostato inteligente.

JavaScript añade funcionalidad e interactivad haciendo que pasen cosas como respuestas a un envento.

Resumen

"Los tres lenguajes son esenciales para el desarrollo web, y cada uno juega un papel crucial en la crecaion de paginas web modernas y funcionales".
